# Welcome to Lexiq

Welcome to the Lexiq documentation! This guide will show you all the different types of content blocks you can use to create rich, engaging documentation.

## Text Content

### Headings and Subheadings

You can use different heading levels to organize your content:

# H1 - Main Heading

## H2 - Section Heading

### H3 - Subsection Heading

#### H4 - Sub-subsection Heading

##### H5 - Minor Heading

###### H6 - Smallest Heading

### Paragraphs and Text Formatting

This is a regular paragraph with **bold text**, _italic text_, and `inline code`. You can also create ~~strikethrough text~~ and <mark>highlighted text</mark>.

Here's another paragraph that demonstrates how text flows naturally. You can create multiple paragraphs to break up your content and make it more readable.

### Lists

#### Unordered Lists

- First item
- Second item with **bold text**
- Third item with _italic text_
- Nested item
  - Sub-item 1
  - Sub-item 2

#### Ordered Lists

1. First step
2. Second step
3. Third step
   1. Sub-step A
   2. Sub-step B

#### Task Lists

- [x] Completed task
- [ ] Pending task
- [ ] Another pending task

### Blockquotes

> This is a blockquote. It's great for highlighting important information, quotes from other sources, or emphasizing key points in your documentation.

> You can also have multiple paragraphs in a blockquote.
>
> And even include **formatting** and `code` within the quote.

## Code and Technical Content

### Inline Code

Use `inline code` for short code snippets, function names, or technical terms.

### Code Blocks

```javascript
// JavaScript code block
function greetUser(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome to Lexiq, ${name}`;
}

const result = greetUser("Developer");
```

```python
# Python code block
def calculate_fibonacci(n):
    if n <= 1:
        return n
    return calculate_fibonacci(n-1) + calculate_fibonacci(n-2)

# Calculate first 10 Fibonacci numbers
for i in range(10):
    print(f"F({i}) = {calculate_fibonacci(i)}")
```

```css
/* CSS code block */
.lexiq-container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 2rem;
}

.lexiq-heading {
  color: #2563eb;
  font-size: 2.5rem;
  font-weight: 700;
  margin-bottom: 1rem;
}
```
